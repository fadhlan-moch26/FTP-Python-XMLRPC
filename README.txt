Cara Menjalankan Program FTP Ini:
1. Reader Harus install Aplikasi VPN LogMeIn HAMACHI atau VPN lainnya
2. Setelah install VPN reader membutuhkan 2 device atau lebih
3. Reader harus menggunakan 1 Device untuk run aplikasi Server dan sisanya untuk client
4. Reader harus mengubah IP addres pada SERVER dan CLient nya (HTTPS://127.....) ubah bagian situ
5. Wajib menjalankan server.py terlebih dahulu lalu client
6. File pada client yang akan di upload ke server harus 1 lokasi dengan client.py
7. Sama hal juga dengan server.
8. Good Luck!
